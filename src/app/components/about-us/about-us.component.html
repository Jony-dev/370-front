BMW system

The system that we have designed, built and implemented for BMW has two main functioning parts. The main part is a GRS system which provides a platform that applicants can apply for jobs at BMW, but further more the platform gives recruiters the power to be able to create a job publication that they can then control effectively and easily. Lastly we have the booking component which gives employees the tools to be able to book a desk for the day at a specific building or to be able to book a boardroom slot
